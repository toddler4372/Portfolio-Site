My Portfolio

Agile

I am accustomed to following an Agile development process when working with teams. I am used to working in tandem with Product Owners, Designers, UX, backend development, and QA. Flowing communication between myself and other members of the team allows me to work efficiently while avoiding needless backtracking which could bottleneck the process.

Creative

I start all my logo art on a sketchpad, and develop site wireframes either on a sketchpad or in Balsamiq. When working on my own, I'll create a style guide for all the elements the page could potentially display. Icon, logo, and other graphical art is created in Illustrator, while Sublime Text, Photoshop and Chrome Dev Tools are used to begin fleshing out the site layout.

Technical

Sublime Text is my editor of choice for adding markup, styles, jQuery and Javascript to my sites. I use JSHint to check all my Javascript for syntax errors that I might miss. Lastly, I love to use Grunt for Javascript tasking such as optimizing and combining CSS files.

Let's Connect!

Quick Bio

I am a Front End Developer living in Redlands, CA. I love creating clean, usable, responsive websites. My passions are family, sports, guitar, and new web technology.
